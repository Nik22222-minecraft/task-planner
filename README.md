# 🚀 NikPlay Task Manager & Telegram Bot

**NikPlay** — это современный веб-планировщик задач с глубокой интеграцией Telegram-бота. Проект помогает удобно организовывать дела, отслеживать прогресс и управлять задачами с любого устройства.

## ✨ Главные фичи (Особенности)
* **Двойной интерфейс:** Управляйте задачами через красивый веб-сайт или прямо на ходу через Telegram-бота.
* **Умное расписание:** Поддержка повторяющихся задач (по дням недели) и автоматическая очистка старых записей.
* **Кастомизация:** 4 визуальные темы (Purple, Green, Sunset, Ocean) и полностью адаптивный мобильный дизайн.
* **Сообщество (Community):** Возможность открыть свой профиль и делиться своими планами с другими.
* **Система заметок:** Продвинутое редактирование заметок (для одной задачи, для всех будущих, для всей категории).

## 🛠 Технологии (Стек)
* **Backend:** Python 3, Flask, SQLAlchemy (SQLite)
* **Telegram Bot:** pyTelegramBotAPI (Telebot) + Webhook
* **Frontend:** HTML5, CSS3, Jinja2
* **Безопасность:** Хеширование паролей (`werkzeug.security`)

## 📦 Установка и запуск (Для разработчиков)

1. Склонируйте репозиторий:
   `git clone `

2. Перейдите в папку проекта:
   `cd nikplay-task-manager`

3. Установите зависимости:
   `pip install -r requirements.txt`

4. Вставьте токен вашего Telegram-бота в файл config.py (переменная BOT_TOKEN).

5. Запустите приложение:
   `python app.py`

*(Примечание: для работы Webhook бота потребуется развернуть приложение на публичном сервере, например PythonAnywhere).*
