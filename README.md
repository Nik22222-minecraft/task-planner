# 🚀 Мультиплатформенный Менеджер Задач (Web + Telegram)

Полноценный full-stack проект для управления личными делами. Позволяет работать с задачами через современный веб-интерфейс и управлять ими на ходу через Telegram-бота. 

## 🛠 Технологии (Стек)
* **Backend:** Python, Flask (Web-сервер)
* **База данных:** SQLite + SQLAlchemy (ORM)
* **Frontend:** HTML5, CSS3 (Градиентные темы, адаптивный дизайн)
* **Интеграции:** Telegram Bot API (работа через Webhooks)
* **Безопасность:** Хеширование паролей (Werkzeug Security)

## 📌 Основной функционал
1. **Регистрация и Безопасность:** Безопасное хранение пользователей. Пароли в базе данных захешированы.
2. **Управление задачами (CRUD):** Создание, редактирование, удаление и отметка о выполнении.
3. **Продвинутая логика:** Умная сортировка (по важности и времени), авто-удаление старых задач, генерация повторяющихся задач на несколько недель вперед.
4. **Кастомизация:** 4 цветовые темы оформления на выбор пользователя.
5. **Сообщество:** Возможность сделать свой профиль публичным и просматривать ленту задач других открытых профилей.

## 🤖 Telegram-бот (@TaskBot_Helper_bot)
Бот построен на принципах ООП (отдельный класс `TaskBotApp`) и работает 24/7 на базе вебхуков (без long-polling).
* `/login [логин] [пароль]` — криптографически безопасная привязка Telegram-аккаунта к сайту.
* `/tasks` — просмотр задач на сегодня с отображением приоритета и времени.
* `/feedback` — сбор отзывов от пользователей (сохраняется на сервере).
* `/report` — отправка сообщений об ошибках разработчику.

## 💻 Как запустить проект локально
1. Клонируйте репозиторий.
2. Установите зависимости: `pip install flask flask-sqlalchemy pyTelegramBotAPI werkzeug`
3. Вставьте свои данные в `app.py` (`SECRET_KEY` и `TOKEN` бота).
4. Запустите файл `app.py`. База данных сгенерируется автоматически.
